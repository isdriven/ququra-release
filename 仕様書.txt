Flash Notes — 仕様書（第一次 fix 時点）
================================================================================

■ 製品コンセプト
  Flash Cards? no, this is Flash Notes!
  マークダウンでノートを書き、伏字で覚える・テストで確認できるデスクトップアプリ。
  グループ／ページ単位で構成し、複数ブックの切り替えに対応。

■ 動作環境
  - デスクトップアプリ（Wails + Vue 3）
  - ブックデータは exe と同じフォルダに置かれたものを読み込む
  - 友達や第三者から受け取ったブックフォルダもそのまま利用可能

--------------------------------------------------------------------------------
1. ブックの取り扱い
--------------------------------------------------------------------------------
  - FILE メニュー
    画面上部の FILE をクリックすると、NEW BOOK・EXPORT・利用可能なブック一覧が表示される。

  - NEW BOOK
    現在のブックを別名でコピーして新規ブックを作成する（例: ququra-book-〇〇）。
    新規ブックを開いた状態になる。

  - EXPORT
    現在のブックを別名でコピーして保存する。バックアップや別用途用のコピーに利用可能。

  - ブックの切り替え
    一覧に表示されたブック名をクリックすると、そのブックに切り替わる。
    編集内容は各ブックごとに保存される。

  - フォント（Option）
    フォントの選択はブックごとに保存される。
    ブック読み込み時に、選択されていたフォントがこの環境にない場合
    （別のPCで使っていたブックや、フォントが削除された場合など）は、未選択に戻る。

--------------------------------------------------------------------------------
2. マークダウン仕様
--------------------------------------------------------------------------------
  編集モードで入力した内容が、閲覧モードでそのまま反映される。

  2.1 見出し
    - # 見出し1 … 上下に 100% 幅の線
    - ## 見出し2 … アクセント色、スタイルそのまま
    - ### 見出し3 … 右側に罫線が伸びる（テキスト＋線）
    - #### 見出し4 … 左側に 20px の太い線

  2.2 その他インライン・ブロック
    - - 項目 … リスト
    - **太字** … 太字
    - --- … 水平線
    - ``` 言語 … コードブロック

  2.3 伏字（redaction）
    - {答え} または {キーワード} のように { } で囲むと、閲覧時に伏字表示。
    - ホバーで正解が表示される。
    - Option の「改行として表示する文字列」で、\n 等を改行として表示可能。

  2.4 テーブル
    - | A | B | の形式。先頭行を数値にすると列幅（%）指定。
    - 合計 100% 未満のときは、テーブル全体の幅もその割合になる。
    - セル内改行は \n または Option の「改行として表示する文字列」で指定。

  2.5 Split（縦割り）
    - <|> … 等分
    - <70|30> … 左 70%、右 30%
    - <80|10|10> … 三分割以上も可能。空は「残り」で按分。

  2.6 同一ページ内リンク（@）
    - @見出し … 同一ページ内の #〜#### の見出しテキストと一致するときだけリンクになる。
    - 一致する見出しがない場合はリンクにせずテキストのみ表示。
    - 別ファイル（他ページ）にはリンクしない。
    - クリック時は Vue Router を経由せず、該当見出しへスクロール。

--------------------------------------------------------------------------------
3. ナビゲーション・UI
--------------------------------------------------------------------------------
  - 左ペイン（nav-pane）
    - 検索バー＋虫眼鏡ボタン（NEW GROUP の上）
      - 入力して検索ボタン押下で全文検索。
      - インクリメンタル検索・自動検索は行わない。
      - 該当するグループ・ファイルに赤ぽち（●）を表示。フィルタリングはしない。
    - NEW GROUP … グループ追加
    - グループ／ページはドラッグ＆ドロップで並び替え・移動可能
    - グループ名・ページ名はダブルクリックで編集

  - メニューバー
    - FILE … NEW BOOK / EXPORT / ブック一覧
    - Option … 改行プレースホルダー、フォント、フォントサイズ
    - Tutorial … マークダウンとブックの取り扱い説明
    - AI Tutorial … AI にテキストを生成させるためのコピペ用プロンプト例（歴史年表・用語集・一問一答・年代順・英単語など）

--------------------------------------------------------------------------------
4. テスト機能
--------------------------------------------------------------------------------
  - テスト作成
    - 閲覧モードで「テスト作成」を押すと、そのページの伏字 { } が入力欄に置き換わったテスト画面が表示される。
    - テストダイアログの横幅は、表示時の main-pane の横幅と等しくする。

  - テスト時・採点時の @ の扱い
    - @見出し はリンクにせず、@xxx ごと削除して表示する（テスト問題・採点結果の両方）。

  - 提出
    - 提出後、採点結果モーダルで正誤と得点を表示。
    - 採点結果モーダルの横幅も、表示時の main-pane の横幅と等しくする。

--------------------------------------------------------------------------------
5. 技術メモ（実装参照用）
--------------------------------------------------------------------------------
  - ルーター: Vue Router hash モード（#/p/:id）
  - 設定・コンテンツ: ブックごとに bbolt に保存
  - パーサーオプション: newlinePlaceholder, validHeadingIds, stripAtLinks（テスト/採点時 true で @ を削除）
  - 目次: getHeadings で #〜#### を取得し、同一ページ内アンカーでスクロール
